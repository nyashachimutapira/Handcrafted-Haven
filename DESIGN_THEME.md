## Typography System

### Font Family: Inter (Sans-serif)
- Modern, clean, highly readable
- Excellent screen rendering
- Good for accessibility

### Typography Hierarchy
```
Heading 1 (H1):  32px - 64px | Bold (700) | Primary color
                 Example: "Handcrafted Haven"

Heading 2 (H2):  24px - 32px | Bold (700) | Primary color
                 Example: "Featured Products"

Heading 3 (H3):  18px - 24px | Bold (700) | Primary color
                 Example: "Shop by Category"

Body Text:       16px | Regular (400) | Dark text
                 Used for descriptions, product info

Small Text:      12px - 14px | Regular (400) | Gray text
                 Used for labels, helper text

Emphasis:        Bold (600) | Primary color
                 Used for important info in body text
```

## Brand Aesthetic

### Key Design Characteristics
1. **Warm & Welcoming**: Brown and tan color scheme creates feeling of natural, handcrafted quality
2. **Artisan-Focused**: Earth tones evoke materials used in crafts (wood, clay, natural fibers)
3. **Trust & Authenticity**: Traditional craftsmanship aesthetic conveyed through color and typography
4. **Clean & Modern**: Sans-serif font and minimal design keep it contemporary
5. **Accessible**: High contrast ratios, clear hierarchy, readable fonts

### Visual Language
- **Icons**: Simple, monochrome brown icons and emoji
- **Images**: Photography of actual handcrafted items
- **Spacing**: Generous whitespace emphasizes quality of each product
- **Shadows**: Subtle shadows create depth without being overwhelming

### Responsive Design
```
Mobile (320px):
Single column layout
Stacked navigation
Buttons full-width
Large touch targets

Tablet (768px):
Two column layout
Side-by-side navigation
Medium buttons
2-column product grid

Desktop (1024px+):
Three+ column layout
Full navigation
Compact spacing
3-4 column product grid
```
